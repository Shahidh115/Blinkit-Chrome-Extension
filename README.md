# Blinkit 👀

Blinkit is a lightweight Chrome extension designed to help users follow the **20-20-20 rule** and reduce digital eye strain during long screen sessions.

## 🚀 Features
- Eye rest reminders
- Simple and clean UI
- Alarm notifications
- Lightweight extension
- Productivity friendly

## 🛠️ Built With
- JavaScript
- HTML
- CSS
- Chrome Extension APIs

## 📖 20-20-20 Rule
Every 20 minutes:
- Look at something 20 feet away
- For 20 seconds

## 📂 Installation
1. Download or clone this repository
2. Open Chrome and go to:
   chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
